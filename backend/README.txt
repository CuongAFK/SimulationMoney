Hướng dẫn chạy dự án

1. Chạy backend server:
   - Lệnh: npm run start
   - Mở terminal mới và chạy lệnh này để khởi động server.js trên port 3000.

2. Tạo Serveo URL cho backend:
   - Lệnh: npm run serveo-backend
   - Chạy lệnh này để tạo và gán URL backend.

3. Chạy frontend Angular:
   - Mở terminal mới và chạy lệnh:
     npm run ng
   - Điều này sẽ mở ứng dụng Angular trên port 4200.

4. Tạo Serveo URL cho frontend:
   - Lệnh: npm run serveo-frontend
   - Chạy lệnh này để tạo URL cho frontend.
